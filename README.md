# Proto

Console commands to work with proto files.

## Install

```shell
composer require quizment/cross-proto --dev
```

## Commands

### Compile proto files

```shell
x docker:compile
```

### Remove all generated PHP files

```shell
x docker:clear
```

### Generate PHP files from proto files

```shell
x docker:generate
```

### Replace long PHP namespace to short

```shell
x docker:replace
```
